# Security Policy

## Supported Versions

The following versions of ScpGUI are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open a public issue. Instead, please report it via the GitHub "Report a security vulnerability" feature or contact the maintainers directly if an email is provided.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
